# Guía de Usuario — Libres Trabaja

Aplicación móvil para la vinculación laboral en Libres, Puebla.

---

## 1. Primeros pasos

### Registro
1. Abre la aplicación
2. Presiona **"Registrarse"**
3. Selecciona tu rol: **Candidato** o **Reclutador**
4. Completa el formulario:
   - Correo electrónico
   - Nombre(s) y apellidos
   - Teléfono
   - Contraseña
5. Presiona **"Registrarse"**
6. ¡Listo! Ahora inicia sesión con tu correo y contraseña

### Inicio de sesión
1. Ingresa tu correo electrónico y contraseña
2. Presiona **"Iniciar Sesión"**
3. La app te llevará a tu pantalla principal según tu rol

---

## 2. Para candidatos

### Explorar vacantes
- En la pantalla de inicio verás **"Vacantes Disponibles"**
- Usa la barra de búsqueda para filtrar por **puesto, sector o empresa**
- Presiona una vacante para ver los detalles completos:
  - Nombre de la empresa
  - Puesto y sueldo
  - Ubicación y sector
  - Descripción del puesto
  - Fotos de las instalaciones (desliza para ver más)

### Postularse a una vacante
1. Presiona una vacante para ver los detalles
2. Presiona **"Confirmar Postulación"**
3. Recibirás una confirmación: "¡Postulación enviada!"
4. La vacante se marcará como aplicada y no podrás volver a postularte

### Ver mis postulaciones
- En el menú inferior, presiona la pestaña **"Postulaciones"**
- Verás todas tus postulaciones activas
- Presiona una postulación para ver detalles
- Puedes eliminar una postulación si ya no te interesa
- Presiona **"Enviar Mensaje"** para chatear con el reclutador

### Gestionar mi perfil
- En el menú inferior, presiona la pestaña **"Perfil"**
- **Foto de perfil**: Presiona el icono de la cámara para cambiarla
- **Editar datos**: Presiona el icono de lápiz junto a cada campo
- **Currículum Vitae**:
  - Presiona **"Subir CV"** para seleccionar un archivo (PDF o imagen)
  - Presiona el CV para previsualizarlo
  - Usa el botón de descarga ⬇ para guardarlo en tu teléfono
  - Presiona el icono de eliminar 🗑 para borrarlo
- **Cerrar sesión**: Presiona el botón rojo al final

### Chat con reclutadores
- Desde la pantalla de postulaciones, presiona **"Enviar Mensaje"**
- Escribe tu mensaje y presiona el botón de enviar
- Los mensajes se actualizan automáticamente cada 2 segundos
- Presiona la foto del reclutador para verla en grande

---

## 3. Para reclutadores

### Publicar una vacante
1. En la pantalla de inicio, presiona el botón **+** (flotante)
2. Completa los campos:
   - **Puesto**: Nombre del cargo
   - **Sueldo**: Salario ofrecido
   - **Sector**: Rubro de la empresa
   - **Ubicación**: Lugar de trabajo
   - **Descripción**: Detalles del puesto
3. Presiona **"Publicar Vacante"**
4. La vacante aparecerá en tu listado

### Ver mis vacantes
- En la pantalla de inicio verás **"Mis Vacantes Activas"**
- Usa la barra de búsqueda para filtrar por puesto
- Cada tarjeta muestra:
  - Nombre de la empresa
  - Puesto ofrecido
  - Número de postulantes
- Presiona **"Ver Postulantes"** para ver quién aplicó

### Editar o eliminar vacantes
1. Desde la pantalla de inicio, presiona **"Mis Vacantes"** en el menú
2. Presiona el icono de lápiz ✏ para editar
3. Presiona el icono de eliminar 🗑 para borrar

### Revisar postulantes
1. Presiona **"Ver Postulantes"** en una vacante
2. Verás la lista de candidatos con:
   - Nombre y correo
   - Foto de perfil (presiona para ver en grande)
   - Puesto al que aplicaron
3. Presiona **"Revisar CV"** para ver su currículum
4. Presiona **"Agendar"** para chatear con el candidato

### Chat con candidatos
- Desde la lista de postulantes, presiona **"Agendar"**
- El chat funciona igual que para los candidatos
- Los mensajes se actualizan automáticamente

### Gestionar mi perfil
- En el menú inferior, presiona la pestaña **"Perfil"**
- **Foto de perfil**: Presiona el icono de la cámara para cambiarla
- **Nombre de empresa, ubicación, teléfono**: Presiona el lápiz para editar
- **Instalaciones**: Presiona **"Agregar Foto"** para subir imágenes de la empresa
  - Presiona una foto para verla en grande
  - Desliza para ver todas las fotos
- **Cerrar sesión**: Presiona el botón rojo al final

## 4. Funcionalidades generales

### Vista previa de CV
- Los CV en **PDF** se renderizan con visor integrado (pdf.js)
- Los CV en **imagen** se muestran con soporte de zoom (pellizcar para ampliar)
- Puedes descargar el CV desde el botón ⬇ en la vista previa

### Zoom en imágenes
- **Pellizca** con dos dedos para hacer zoom
- **Arrastra** con un dedo cuando estés ampliado
- El zoom se reinicia al abrir una nueva imagen

### Notificaciones
- Recibirás notificaciones push cuando recibas un nuevo mensaje
- Asegúrate de tener una conexión a internet activa

## 5. Solución de problemas

| Problema | Solución |
|----------|----------|
| No carga la lista de vacantes | Verifica tu conexión a internet |
| Error 401 al previsualizar CV | Cierra y vuelve a abrir el CV — se usará una URL firmada |
| No se envía el mensaje | Verifica tu conexión; el mensaje se reintenta automáticamente |
| La foto no se actualiza | Espera unos segundos y recarga la pantalla |
| La app se cierra sola | Asegúrate de tener Android 7.0 o superior |
| No recuerdo mi contraseña | Contacta al administrador del sistema |
